# WP1111_Final

NTU Web Programming Final Project

## Local Test & Install

### Frontend

In frontend/ `yarn` to install modules,
and `yarn start` to open frontend

### Backend

In backend/ `yarn` to install modules,
and `yarn server` to open server and connect to DB
